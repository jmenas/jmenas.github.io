📋 Google Sheet Setup

How to connect: In Google Sheets, go to File → Share → Publish to web, choose Comma-separated values (.csv), then click Publish. Copy the generated link and paste it above. Or use the export URL format: https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/export?format=csv

Expected columns (in order): Title · Author · Genre · Pages · Publication Year · Date Read · Rating (1-5) · Status — Status options: Read, Reading, Up Next, Want to Read
